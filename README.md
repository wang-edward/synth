how to get zig 0.15.1
```
brew install asdf
asdf plugin add zig https://github.com/asdf-community/asdf-zig.git
asdf install zig 0.15.1
asdf set zig 0.15.1
zig env
```

build
```
zig build
```

run
```
zig build run
```
